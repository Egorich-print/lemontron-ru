## Lemontron Build Steps

1. Order all the off-the-shelf parts from the BOM.
2. Print the plastic parts.
3. Modify some off-the-shelf parts.
4. Assemble the Lemontron.
5. Install the software.
6. Calibrate the Z-offset.
